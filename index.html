Om navneord – i nomi

Køn: Hvert navneord er enten hankøn eller hunkøn. Tommelfingerregel:

  • Ender på -o → ofte hankøn: il libro (bogen)
  • Ender på -a → ofte hunkøn: la casa (huset)
  • Ender på -e → kan være begge – skal læres udenad: il fiore (m), la notte (f)

Flertal: -o → -i, -a → -e, -e → -i. Eks.: libro → libri, casa → case, fiore → fiori.

Om tillægsord – gli aggettivi

Tillægsord (adjektiver) retter sig efter navneordet i køn og tal. Som regel står de efter navneordet: una casa rossa (et rødt hus).

Bemærk: tillægsord på -e skelner kun mellem ental og flertal, ikke køn.
